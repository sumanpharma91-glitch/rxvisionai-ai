RxVision AI — OCR + AI Extraction

Files:
- index.html
- mvp.html
- api/extract.js
- package.json

Vercel setup:
1. Add OPENAI_API_KEY in Project Settings > Environment Variables.
2. Optional: add OPENAI_MODEL. Default is gpt-5.5.
3. Deploy extracted files, not the ZIP itself.
4. Open /mvp.html.
5. Run OCR, then click Clean with AI.

Important: Never put the API key in mvp.html.
